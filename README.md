# OpenCV-ile-renk-tespiti
Python dilinde HSV uzayında renk ve belirlenen rengin koordinatları
Bu projede HSV renk uzayında alçak ve yüksek renk aralıkları belirlenerek görüntü üzerinden maskeleme işlemi yapılmaktadır. 
Yapılan maskeleme işleminin ardından contour hesaplaması yaparak rengin koordinatları ve etrafında isteğe göre circle isteğe göre rectangle çizilmektedir.
